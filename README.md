A simple **Grocery Store Management System** built using **Python (FastAPI)** for the backend, **MySQL** for the database, and **HTML, CSS, JavaScript (jQuery + Bootstrap)** for the frontend.

This project is created for **learning full-stack development basics**, including CRUD operations, API integration, and Git usage.

---
 Features

  Products Module
  ---------------------------
- View all products
- Add new products
- Delete products (soft delete using `is_active`)
- Products linked with UOM (Unit of Measurement)

  Orders Module
  -----------------------------
- Create new orders
- View all orders
- Order details stored in database

  UOM Module
  ----------------------------
- Fetch and display available units (Kg, EACH, etc.)
